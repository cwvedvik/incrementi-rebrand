# Incrementi — Rebrand & Repositioning

Foundational assets for the rebrand and repositioning of **incrementi.no** — an AI-led
transformation partner for industrial, maritime and financial firms in the Nordics.
Part of 99X Group.

> **The compounding advantage** — the operating system for the AI-native firm.

## Contents

| File | What it is |
|------|------------|
| `Incrementi_Brand_Positioning_Strategy.html` | The source-of-truth brand & positioning strategy: category, positioning, messaging pillars, service model (the Incrementi Journey), identity, website architecture, content pillars, roadmap. |
| `Incrementi_Web_Experience_Prototype.html` | Working concept prototype of the "living intelligence" homepage — a conversational, brain/organism web experience with content delivered on demand. |
| `v1/` | Locked v1 snapshots (2026-07-01). |

Both files are self-contained HTML — open directly in a browser.

## Visual system

- Canvas: near-black `#08090B` / deep navy `#0C1420`
- Text: warm off-white `#F4F2EE`
- Accent: metallic copper `#C98A5E`, secondary steel `#8FA3B8`
- Type: Fraunces (display serif) + DM Sans (body/UI)
